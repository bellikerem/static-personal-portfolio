Hakkımda

Benim adım Kerem Can Belli. Bahçeşehir Üniversitesinde 1. sınıf bilgisayar mühendisliği öğrencisiyim. Web geliştirme ve yapay zeka alanlarına ilgi duyuyorum. Liseyi Prof. Dr. Necmettin Erbakan Fen Lisesinde okudum ve 97.6 not ortalamasıyla mezun oldum ve BAU'daki eğitimime %100 bursla kabul edildim. Şu anda Python (Nesne Tabanlı Programlama, algoritmalar) ve Web geliştirme (HTML5, CSS3) konularında kendimi geliştiriyor aynı zamanda Git ve Github gibi profesyonel araçları sürüm kontrolü için aktif olarak kullanıyorum. Deneyimli geliştiricilerden öğrenmeye, birlikte projeler üretmeye ve yapay zeka ile web teknolojilerini birleştiren yenilikçi çalışmalara katkı sağlamaya istekliyim.
